部署Hexo至GitHub

  • 修改配置文件
    打开MyBlog文件夹下面的_config.yml 文件进行修改补充
    找到以下字段尽行修改

#site
title: XXX的博客
subtitle: ‘My personal blog’
description: ‘记录、分享、交流’
keywords:
author: 李华
language: zh-CN
timezone: ‘’

#URL
##Set your site url here. For example, if you use GitHub Page, set url as ‘https://username.github.io/project'
url: https://username.github.io/
permalink: :year/:month/:day/:title/
permalink_defaults:
pretty_urls:
trailing_index: true # Set to false to remove trailing ‘index.html’ from permalinks
trailing_html: true # Set to false to remove trailing ‘.html’ from permalinks

#Deployment
##Docs: https://hexo.io/docs/one-command-deployment
deploy:
type: git
repository: ==git@github.com:username/username.github.io.git #F44336==
branch: ==main #F44336==
注意标红部分，repository的内容可在自己的GitHub仓库中进行复制，有两种形式：（1）没有配置则使用Https形式的地址
（2）配置SSH Key之后的git@github.com:xxx形式
